Overview

This capstone project explores the use of pose estimation and sequential deep learning to predict penalty kick direction. The model classifies kicks as left, center, or right, based solely on pre-kick body movement. With over 500 real-world penalty clips processed and analyzed, this system aims to provide goalkeepers with an edge through predictive motion cues.

Methodology

  • Collected 560 penalty clips (2015â€“2024), segmented to 3â€“5 seconds before the kick.
  • Used Roboflow to label the kicker and train a YOLOv8 detection model.
  • Applied YOLOv8 pose estimation to extract 17 keypoints across 30 frames per clip.
  • Trained a Long Short-Term Memory (LSTM) neural network on sequences of 34 keypoint values to classify direction.

Technology Stack

  • Python: Data processing, modeling, and visualization
  • YOLOv8: Custom object detection and pose estimation
  • TensorFlow/Keras: LSTM model for sequential prediction
  • Adobe Premiere Pro: Manual segmentation of video clips
  • Roboflow: Frame annotation and model training interface

Highlights

  • Model accuracy reached 41.5%, outperforming published academic baselines.
  • Identified key joints (hip, ankle, wrist) most predictive of direction.
  • Built interpretable decision tree to trace model logic from keypoints.
  • Designed a pipeline suitable for real-time goalkeeper feedback in training.
